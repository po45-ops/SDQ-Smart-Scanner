# SDQ Smart Scanner

เว็บแอปช่วยบันทึก จัดการ ตรวจทาน และส่งออกคะแนนแบบประเมินพฤติกรรมนักเรียน SDQ

## ฟังก์ชันหลัก

- สมัครและเข้าสู่ระบบบัญชีคุณครู
- เพิ่มรายชื่อนักเรียนทีละคนหรือนำเข้าจากข้อมูลที่คัดลอกจาก Excel
- แยกรายชื่อตามชั้น/ห้อง
- อัปโหลดหรือถ่ายภาพแบบประเมิน
- อ่านภาพด้วย Gemini API แบบใช้ API Key ของผู้ใช้
- กรอกและแก้คะแนนทั้ง 25 ข้อด้วยตนเอง
- คำนวณคะแนนสรุป 5 ด้าน
- ส่งออกตารางเป็นไฟล์ Excel
- รองรับมือถือ แท็บเล็ต และคอมพิวเตอร์

## การจัดเก็บข้อมูล

เวอร์ชัน GitHub Pages เก็บบัญชี รายชื่อนักเรียน และคะแนนไว้ใน Local Storage ของเบราว์เซอร์เครื่องที่ใช้งาน ข้อมูลไม่ซิงก์ข้ามเครื่องและไม่ควรใช้งานบนอุปกรณ์สาธารณะ รหัสผ่านจัดเก็บเป็นค่า SHA-256 ไม่ได้จัดเก็บเป็นข้อความตรง

Gemini API Key เก็บเฉพาะ Session Storage ของแท็บปัจจุบัน เมื่อกดวิเคราะห์ ภาพแบบประเมินจะถูกส่งตรงไปยัง Gemini API

## พัฒนาในเครื่อง

```bash
npm install
npm run dev
```

สร้างไฟล์สำหรับเผยแพร่:

```bash
npm run build
```

> ระบบช่วยคีย์และคำนวณคะแนนเบื้องต้น ไม่ใช่เครื่องมือวินิจฉัย ควรตรวจเทียบกับแบบประเมินต้นฉบับก่อนบันทึกทุกครั้ง
